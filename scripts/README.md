# Scripts · Groovy

Esta carpeta contiene los scripts Groovy utilizados en los iFlows de SAP Integration Suite.

## Contenido

- `ValidateH2Payload.groovy` → Validación del payload del sensor IoT
- `CalculateCO2Savings.groovy` → Cálculo de CO₂ ahorrado vs hidrógeno gris
- `GenerateGreenH2Certificate.groovy` → Generación del certificado EU-GreenH2
