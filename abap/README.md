# ABAP Cloud · S/4HANA

Esta carpeta contiene los objetos ABAP Cloud desarrollados en Eclipse ADT para SAP S/4HANA Cloud.

## Contenido

- `ZI_H2Production.cds` → CDS View de producción de hidrógeno verde
- `ZR_H2ProductionTP.cds` → CDS View de transporte RAP
- `ZBP_H2Production.abap` → Behaviour Implementation del RAP Business Object
- `ZI_H2Production.xml` → Metadata Extension Fiori

## Requisitos

- Eclipse ADT con conexión a SAP S/4HANA Cloud Trial
- Paquete ABAP Cloud habilitado
